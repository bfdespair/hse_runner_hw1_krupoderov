# Минималистичный раннер (HomeAssignment_1)

Круподеров Егор БПМИ229

Описание игры

Минималистичный 3D раннер. Игрок управляет кубом, который:
- автоматически движется вперёд;
- может перемещаться влево и вправо по трём линиям;
- может выполнять до двух прыжков (двойной прыжок);
- имеет запас здоровья и получает урон при столкновении с препятствиями;
- при смерти игра перезапускается.

На пути игрока:
- периодически генерируются препятствия двух типов с разным уроном;
- случайно выбирается линия (лево/центр/право);
- старые препятствия и бонусы удаляются, когда игрок достаточно далеко.

Точки входа

- Главная сцена: MainScene (добавлена в Build Settings).
- Главный игровой скрипт игрока: PlayerController.cs.
- Управление потоком игры и сложностью: GameManager.cs.
- Генерация препятствий и бонусов: ObstacleSpawner.cs.

Выполненные дополнительные требования

1. Два типа бонусов и больше:
   - Heal — восстанавливает здоровье игрока.
   - Speed — временное ускорение игрока.
   - Invulnerability — временная неуязвимость.

2. Ускорение игры с течением времени:
   - Параметр сложности увеличивается каждые difficultyIncreaseInterval секунд.
   - Скорость движения игрока умножается на difficultyMultiplier.

Все важные параметры (скорость, максимальное здоровье, урон препятствий, частота генерации, длительность и сила бонусов) задаются через Inspector у соответствующих компонентов:
- PlayerController
- Obstacle
- ObstacleSpawner
- GameManager
- Bonus

- Документация Unity:
  - Physics, Rigidbody
  - Input (старый Input System)
  - SceneManager
